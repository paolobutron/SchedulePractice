Este sistema permite:
  • Calcular los montos correspondientes a DAF y DUF para Terminales, Radiobases, Bandas de Frecuencias y Radioenlaces de la red de Nuevatel, sean mensuales, anuales o a demanda; para fines de declaraciones y pagos regulatorios a la ATT.
  • Contar con una Base de Datos de Radiobases (Sitios), y su permanente actualización a través de altas, bajas y modificaciones.
  • Contar con una Base de Datos de Radioenlaces, y su permanente actualización a través de altas, bajas y modificaciones.
La administración de este sistema está a cargo de las Vicepresidencias de Regulación e IT (contacto Regulación: moira.diezdemedina@nuevatel.com , contacto IT: hugo.ona@nuevatel.com)
Sistema desarrollado In-house.
